## [0.2.1] - 2026-04-13

### Added

- Better chat rendering with Markdown tables, task lists, LaTeX, Mermaid, and chart support.
- Session group collapse state persistence.
- Improved provider model management with remote model selection modal and automatic saving feedback.

### Changed

- Reduced VSIX package size and streamlined the chat runtime and refresh pipeline.
- Improved streaming responsiveness, optimistic UI updates, and provider/model configuration workflows.
- Refactored repository, provider client, controller, and state modules for better maintainability.
