## 🚀 快速开始

### 安装

- **VS Code**: 在应用商店搜索 "ChatBuddy"，或[在线安装](https://marketplace.visualstudio.com/items?itemName=Zheng404.chatbuddy)
- **VSCodium/其他**: [从 Open VSX 安装](https://open-vsx.org/extension/Zheng404/chatbuddy)
- **手动安装**: 从 [GitHub Releases](https://github.com/Zheng404/ChatBuddy/releases) 下载 VSIX 文件，选择从 VSIX 安装

### 配置

1. 打开 VS Code 侧边栏的 ChatBuddy 面板
2. **模型配置** — 配置 AI 服务商和 API 密钥
3. 在助手列表中新建一个助手
4. 开始对话！
